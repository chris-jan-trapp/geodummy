Hello There!

This is a minimal implementation that acts as a geoserver for TESTING PURPOSES ONLY! Seriously, DO NOT USE THIS ANYWHERE NEAR PRODUCTION. No single component is robust or has a tendency to fail in a safe manner. Even the webserver is a development server and runs in DEBUG MODE.

This thing has been build with flask and provides a minimal feature set. There's this page ad the end point features/ that either expects POST with json containing 

{"features": [...]}
or GET. POST will store the features and return the number of stored items and GET will return all stored features. NO CHECKING IS PERFORMED! If your POST has features, they will be stored.
